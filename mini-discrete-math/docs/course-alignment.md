# Course Alignment: MIT 6.042J Mathematics for Computer Science

## Module ⬄ Course Mapping

| This Module | MIT 6.042J Section | Topics |
|-------------|-------------------|--------|
| `logic`     | Unit 1: Proofs    | Propositional logic, truth tables, tautology, contradiction, induction |
| `number`    | Unit 2: Number Theory | GCD, extended Euclidean, modular arithmetic, RSA, prime factorization |
| `set`       | Unit 3: Sets      | Set operations, power set, Cartesian product, cardinality |
| `relation`  | Unit 3: Relations | Binary relations, properties, closures, equivalence, partial orders |
| `graph`     | Unit 4: Graph Theory | BFS/DFS, shortest paths, topological sort, coloring, Euler circuits |
| `tree`      | Unit 5: Trees     | BST, AVL, tree traversals, Huffman coding |
| `combo`     | Unit 6: Counting  | Permutations, combinations, Stirling numbers, Catalan, partitions, derangements |
| `boolalg`   | Extension: Digital Logic | Truth tables, SOP/POS, Karnaugh maps, logic circuits |
